# crux/homebrew-tap

Homebrew cask tap for [crux](https://github.com/crux)'s macOS apps.

## Available casks

- `schlonk-pad` — stable channel
- `schlonk-pad-dev` — dev builds from `main`

## Install

```
brew install --cask crux/tap/schlonk-pad
# or for dev:
brew install --cask crux/tap/schlonk-pad-dev
```

The cask files here are written automatically by the source repo's release
workflows. Don't edit them by hand — your changes will be overwritten on the
next release.
